# Contributing to QR Code Maker

Thanks for your interest in contributing! A few guidelines:

- Please open an issue before submitting larger changes so we can discuss design.
- Fork the repository and work on a feature branch named `feature/your-feature`.
- Keep changes focused and include tests where applicable.
- Follow existing code style and run any formatters if present.
- Submit a pull request describing the change and link related issues.

## Code of Conduct

Be respectful and constructive. Treat contributors with courtesy.
